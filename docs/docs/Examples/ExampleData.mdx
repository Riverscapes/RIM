All of the 37 sites reported in Bartelt ([2021](https://digitalcommons.usu.edu/etd/8226)) are packaged into [**riverscapes projects** ](https://riverscapes.net/Tools/Technical_Reference/Documentation_Standards/Riverscapes_Projects/), which can be viewed in the Web Riverscapes Viewer, or [Riverscapes Viewers](https://viewer.riverscapes.net) in your GIS. You will need a [free Riverscapes Consortium account to view](https://riverscapes.net/).

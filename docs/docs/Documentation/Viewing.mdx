### Install Riverscapes Viewer for your Desktop GIS

If you want to view a RIM project in desktop GIS, they are designed to work well with Riverscapes Viewer (RV). Follow the instructions on the [Viewer install page](http://viewer.riverscapes.net) to install the RAVE toolbar for QGIS or ArcGIS. It takes less than a minute.
